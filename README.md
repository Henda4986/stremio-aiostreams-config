## 📦 What is stremio-aiostreams-config?

This repository contains an optimized setup for streaming movies, series, and anime using Stremio. It uses two popular add-ons:

- **AIOStreams** – for finding streams fast  
- **AIOmetadata** – for fetching detailed metadata

The configuration prioritizes 1080p streams and detects REMUX files for better quality. It also helps skip anime filler episodes and keeps the interface clean and simple.

This setup works well with **Torrentio** and various **Debrid services** to improve stream availability and speed.

## 🔍 How the Setup Works

This config orders streams roughly like this:

```
1080p REMUX
1080p Bluray
1080p WEB-DL
2160p REMUX
2160p Bluray
720p WEB
```

The 1080p streams come first to give you faster loading and smoother playback. Most TVs upscale 1080p well, so you get quality without lag.

REMUX files are detected to provide better audio and video quality compared to normal releases.

## 📊 Stream Card Example

Here is how a typical stream card looks in Stremio when using this config:

```

Oppenheimer
REMUX HEVC
Atmos TrueHD 7.1 | EN
68.4 GB | 62 Mbps | SUB
```

This card shows:

- Source type: REMUX, Bluray, or WEB  
- Video codec (HEVC in this example)  
- Audio codec and channels (7.1 Audio here)  
- Audio language  
- File size and bitrate  
- Subtitle availability

## ⚙️ Recommended Setup Tips

- Enable Debrid services like Real-Debrid or AllDebrid within Stremio to unlock more streams and faster buffering.  
- Make sure your internet connection is stable at least 20 Mbps for smooth 1080p playback.  
- Update your graphics drivers if you plan to watch 4K content.  
- Test a few shows or movies to confirm the setup works before regular use.  

---

## 🛠️ Troubleshooting

- If streams are slow or fail to load, check your internet connection.  
- Reload the configuration file if you see outdated or missing streams.  
- Ensure your Debrid service is connected and active in Stremio settings.  
- Restart Stremio to refresh the stream list if it hangs or crashes.
